## Funcionalidades
Se requiere que este microservicio tenga las siguientes funciones:

- Crear un item, atributos necesarios:
  - id (long)
  - name (string)
  - description (string)
  - price (double)
  - quantity (long)
- Buscar por nombre
- Actualizar un producto por id
- Mostrar todos los productos
- Buscar producto por id.
